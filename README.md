## ğŸš€ Commands

### Study Commands
- `!study [subject] [duration]` - Start a study session (default: 30 min)
- `!breaktime [duration]` - Take a timed break (default: 5 min)
- `!todo add [task]` - Add a task to your to-do list
- `!todo list` - View your to-do list
- `!todo done [number]` - Mark a task as complete

### Timer Commands
- `!timer [minutes]` - Set a custom study timer
- `!pomodoro` - Start a 25-minute Pomodoro session

### Other Commands
- `!motivate` - Get a motivational quote
- `!help` - Display all available commands
- `@Venus Bot` - Mention the bot to see the help menu
